Entrega del TP Final de Miguel Aponte. Arasunu App. Otoño 2025 con el profesor Diego Bonnin

Manual de uso - Arasunu App
Inicio de sesión:
1. Al abrir la app, ingrese sus credenciales para acceder.

Pantalla principal (Home):
1. Desde aquí puede navegar a las distintas secciones: Pronóstico, Configuración o Información.

Consultar el clima:
1. Ingrese el nombre de una ciudad en la pantalla de pronóstico.
2. Presione el botón para obtener el clima actual.
3. Se mostrará la temperatura y una breve descripción del clima.

Configurar unidades:
1. En la pantalla de configuración puede elegir entre Celsius o Fahrenheit.

Información:
1. La sección "Acerca de" muestra información sobre la aplicación y su propósito.
